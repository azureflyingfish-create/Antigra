# メンバー管理システム 操作ガイド

## 概要
本システムは、プロジェクトとメンバーのアサイン管理を行うアプリケーションです。
Dockerを用いて以下の技術スタックで構築されています。

- **Frontend**: Vue.js + Element Plus
- **Backend**: Python (FastAPI)
- **Database**: PostgreSQL

## 起動方法

### 前提条件
- **OrbStack** (または Docker Desktop) がインストールされ、起動していること。

### 手順
1. ターミナルでプロジェクトルート（`docker-compose.yml`があるフォルダ）を開きます。
2. 以下のコマンドを実行してシステムを起動します（OrbStackがdockerコマンドを提供します）。
   ```bash
   docker compose up --build
   ```
3. 初回起動時はデータベースの初期化やパッケージのインストールが行われるため、数分かかる場合があります。

## アクセス方法
起動後、ブラウザで以下のURLにアクセスしてください。

- **アプリケーション**: [http://localhost:8080](http://localhost:8080)
- **API ドキュメント (Swagger)**: [http://localhost:8000/docs](http://localhost:8000/docs) (Backend直接アクセス時)

## 機能一覧
1. **ダッシュボード**: 全体の統計情報やタイムライングラフを表示します。
2. **メンバー管理**: メンバーの追加・編集・削除が可能です。
3. **プロジェクト管理**: プロジェクトの期間設定や説明の追加が可能です。
4. **設定**: 役職（タイトル）のカスタマイズが可能です。

## トラブルシューティング
- **起動しない場合**: ポート8080や8000が既に使用されていないか確認してください。
- **データが保存されない**: Docker Volume (`postgres_data`) が正しくマウントされているか確認してください。
